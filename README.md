## Compilation
To compile the program, use the following command:

```bash
gcc -O3 -Wall -Wextra -Werror -Wno-unused-result -o proj1 *.c
```
## Run
Run the program using the following command:

```bash
./proj1
```

## Testing
To run all tests:
1. cd into folder containing all tests
```bash
cd tests
```
2. Run command make:
```bash
make
```
